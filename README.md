# EduLearn – Online Learning Platform

EduLearn is a modern web-based learning platform that enables students to access courses, track progress, complete assignments, and interact with instructors from anywhere.

## Features

### For Students
- User Registration and Login
- Course Enrollment
- Video Lessons and Learning Materials
- Assignment Submission
- Progress Tracking
- Certificate Generation
- Quiz and Assessment System
- Personalized Dashboard

### For Instructors
- Course Creation and Management
- Upload Videos and Learning Resources
- Create Quizzes and Assignments
- Student Performance Monitoring
- Course Analytics

### For Administrators
- User Management
- Course Management
- Content Moderation
- Platform Analytics
- Role-Based Access Control

## Technologies Used

- React.js

## Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/edulearn.git
```

2. Install dependencies

```bash
npm install
```

3. Run the application

```bash
npm run dev
```
